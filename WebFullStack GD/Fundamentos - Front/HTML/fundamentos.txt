++++O cliente solicita, o servidor oferece uma resposta para a requisição do liente.
----------------------------------------------

HTML(Hyper text markup language):
     O esqueleto do Site, é baseado em TAGS
JS: 
    Traz interação para o site (Análago ao corpo humano, representa os musculos)-Linguagem de alto nivel
CSS: 
    Estiliza o site (Funciona a base de seletor, valor)

-----WEB SITE X APLICAÇÕES WEB----------------
++++Site:
    Possuem menos interações, são mais estásticos: Nem todo site é uma aplicação web.
++++Aplicações:
    Mais dinamico, mais interações, mais solicitações e executa mais processos. Hoje em dia, trabalhamos mais com aplicações web.
    EX:
        Um site de compras(Carrinho, validação, dados) As aplicações usam a arquitetura de cliente servdor. O servidor permite que o cliente interaja com a aplicação sem precisar baixar nada. 
        Meet, gmail,drive, aplicações de streaming...
